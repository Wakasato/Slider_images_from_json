    About This Code

    What would you do in this situation?
    Your customer requests that they want update website's slider images without using html or CSS.
    But You can't use PHP because of their server's restriction.
    My works have kind of that problems sometimes.

    This kind of restriction apply to Japanese server service like "Rakuten Gold".
    In this situation, I have to use javascript and json.
    In Generally, customers edit json files which have data of image's name, file's location, and title.
    After that, they upload only json files with FTP apps.
    It is not difficult for customer who can use Excel or Word.

    So, I wrote javascript that import json data, change DOM, execute slider moving function.
    It have call back function to allow slider function executing after
    changed DOM including json data.
    It is little complicated and tirign.
    I don't want write this code twice.
    In addition, I thought that someone need this codes in this huge world.
    For that reason, I upload this code.

    このコードについて

    このような場合、あなたならどうしますか？
    あなたの顧客から、htmlやCSSを使わずにWebサイトのスライダー画像を更新したいという要望がありました。
    でも、サーバーの制限でPHPが使えません。
    私の仕事では、こういう話が時々あります。

    このような制限は、「楽天ゴールド」のような日本のサーバーであります。
    こういう場合、javascriptとjsonを使わなければなりません。
    一般的に、お客様は画像の名前、ファイルの場所、タイトルなどのデータが入ったjsonファイルを編集します。
    その後、FTPアプリでjsonファイルだけをアップロードします。
    エクセルやワードを使えるお客様にとっては難しくありません。

    そこで、jsonデータを取り込み、DOMを変更し、スライダーを動かす機能を実行するJavascriptを書きました。
    jsonデータを含むDOMを変更した後にスライダー関数を実行できるように、コールバック関数があります。
    ちょっと複雑で面倒す。
    このコードを2回は書きたくないです。
    また、このコードは色い世界で誰かに必要とされているのではないかと思いました。
    そのために、このコードをアップロードしました。
